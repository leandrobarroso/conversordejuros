Conceitos básicos

Juro é a remuneração ou o custo do capital, podendo ser também definido como o "aluguel pago pelo uso do dinheiro".

Sendo o juro o aluguel do capital, ele deve ser calculado de forma diretamente proporcional ao capital e ao prazo.

Sob o sistema de capitalização composta (taxas efetivas), duas taxas são equivalentes se aplicadas ao mesmo capital, durante o mesmo período de tempo, produzem o mesmo montante (capital + juros).

O Conversor de juros é uma ferramenta que permite converter a taxa de juros referente a qualquer periodicidade de tempo em uma taxa equivalente em outra periodicidade. Por exemplo, você pode converter a taxa de juros anual para mensal ou diária.

Fórmulas utilizadas para conversão de taxas de juros

  • Diária para mensal

    \(i_m = (1 + i_d)^{30}-1\)

  • Diária para anual

    \(i_a = (1 + i_d)^{360}-1\)

  • Mensal para diária

    \(i_d = (1 + i_m)^{1/30}-1\)

  • Mensal para anual

    \(i_a = (1 + i_m)^{12}-1\)

  • Anual para diária

    \(i_d = (1 + i_a)^{1/360}-1\)

  • Anual para mensal

    \(i_m = (1 + i_a)^{1/12}-1\)

Sendo:

  • \(i_d\) = taxa de juros diária
  • \(i_m\) = taxa de juros mensal
  • \(i_a\) = taxa de juros anual
